## Heap Tree Recipes
- ### **Clean** - Removes Build Output
    ```
    $ make

    or

    $ make clean
    ```

- ### **basic** - Creates a min-heap from a given set of integers
    ```
    $ make basic
    ```

- ### **sort** - Creates a max-heap from a given set of integers, sorts it using heap sort
    ```
    $ make sort
    ```
